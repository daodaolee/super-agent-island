# SuperAgentIsland v0.0.6 Release Archive

Date: 2026-06-29

## Scope

- Fixed the GAC release-secret migration fallback introduced in `v0.0.5`.
- Existing local secrets now fall back to the active second GAC account slot,
  not the retired first slot.
- Kept the preferred new release secret names
  `GAC_IT_SERVICE_EMAIL` and `GAC_IT_SERVICE_PASSWORD`.
- Updated landing page download links, Homebrew Cask metadata, changelog, and
  release archive notes for `v0.0.6`.
