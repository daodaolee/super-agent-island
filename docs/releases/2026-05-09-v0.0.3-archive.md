# SuperAgentIsland v0.0.3 Release Archive

Date: 2026-05-09

## Scope

- Added Feishu OAuth login for SuperAgent through an in-app WebKit login window.
- Persisted validated SuperAgent login state at runtime instead of embedding
  SuperAgent credentials in release builds.
- Redesigned the Account settings interaction so password login and Feishu login
  are mutually exclusive until logout.
- Updated the landing page download links and release display to `v0.0.3`.

## Notes

- SuperAgent password login now uses settings-time credentials.
- Feishu login uses the SuperAgent web OAuth entrypoint and validates
  `/api/v1/auth/me` before recording the session as logged in.
- GAC release secrets remain configured through `.release-secrets.env`.
