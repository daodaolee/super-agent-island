三步安装。

同事只需要下载 DMG，把应用拖到 Applications，然后在设置里填写 SuperAgent Web 账号即可。

01

下载 DMG

从 GitHub Release 下载最新的 SuperAgentIsland 安装包。

02

拖到 Applications

打开 DMG 后把 App 拖入 Applications，按 macOS 提示允许运行。

03

测试账号

在账号设置里填写 SuperAgent Web 账号，点击测试确认可用。

准备安装 SuperAgentIsland。

当前公开版本 v0.0.1，后续版本会通过 Sparkle 自动检查更新。
